## Deploy

A site está hospedada na plataforma Vercel e pode ser acessada neste [link](https://erickcreis-proffy.vercel.app/).

## Como rodar localmente

```bash
# Vá para a pasta web
$ cd web

# Instale as depedencias
$ yarn install

# Rode a aplicação
$ yarn start
```
Acesse: http://localhost:3000/
